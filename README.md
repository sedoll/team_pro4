# 📂 프로젝트 소개 
​
안녕하세요! <br/>
**Haebeop**은 학생들을 위한 온라인 강의 플랫폼으로,<br/>
교육에 관한 정보를 공유하고 상호 지원하는 공간을 제공합니다.  
​<br/><br/>
학생들의 성장과 교육 과정을 더욱 효과적으로 관리하고,<br/> 
경험을 나누며, <br/>유익한 자원을 활용하여
 **함께 성장할 수 있는 플랫폼을 제공**하는 것이   
 저희 팀의 기획의도 입니다.  
 <br/> <br/>
